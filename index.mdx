## Quick Start

Get up and running with TanStack Query in your preferred framework in minutes.

<Steps>
  <Step title="Install TanStack Query">
    Choose your framework and install the corresponding package:

    <CodeGroup>
    ```bash npm (React)
    npm install @tanstack/react-query
    ```
    ```bash yarn (React)
    yarn add @tanstack/react-query
    ```
    ```bash pnpm (React)
    pnpm add @tanstack/react-query
    ```
    ```bash npm (Vue)
    npm install @tanstack/vue-query
    ```
    ```bash npm (Solid)
    npm install @tanstack/solid-query
    ```
    ```bash npm (Svelte)
    npm install @tanstack/svelte-query
    ```
    ```bash npm (Angular)
    npm install @tanstack/angular-query-experimental
    ```
    </CodeGroup>
  </Step>

  <Step title="Set up the Query Client">
    Initialize the QueryClient and wrap your application with the provider:

    ```tsx React
    import { QueryClient, QueryClientProvider } from '@tanstack/react-query'

    const queryClient = new QueryClient()

    function App() {
      return (
        <QueryClientProvider client={queryClient}>
          {/* Your app components */}
        </QueryClientProvider>
      )
    }
    ```
  </Step>

  <Step title="Create your first query">
    Use the query hook to fetch and cache data:

    ```tsx React
    import { useQuery } from '@tanstack/react-query'

    function Todos() {
      const { data, isLoading, error } = useQuery({
        queryKey: ['todos'],
        queryFn: () => fetch('/api/todos').then(res => res.json())
      })

      if (isLoading) return <div>Loading...</div>
      if (error) return <div>Error: {error.message}</div>

      return (
        <ul>
          {data.map(todo => (
            <li key={todo.id}>{todo.title}</li>
          ))}
        </ul>
      )
    }
    ```

    <Note>
    TanStack Query automatically handles caching, background refetching, and data synchronization out of the box!
    </Note>
  </Step>
</Steps>
