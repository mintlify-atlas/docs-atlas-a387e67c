---
title: QueryClient
description: The QueryClient manages queries and mutations for TanStack Query.
---

The `QueryClient` can be used to interact with a cache and provides methods to fetch, invalidate, and manage queries and mutations.

## Constructor

```typescript
const queryClient = new QueryClient(config?: QueryClientConfig)
```

<ParamField path="config" type="QueryClientConfig" optional>
  Configuration object for the QueryClient
  
  <ParamField path="config.queryCache" type="QueryCache" optional>
    Custom QueryCache instance. If not provided, a new QueryCache will be created.
  </ParamField>
  
  <ParamField path="config.mutationCache" type="MutationCache" optional>
    Custom MutationCache instance. If not provided, a new MutationCache will be created.
  </ParamField>
  
  <ParamField path="config.defaultOptions" type="DefaultOptions" optional>
    Default options for queries and mutations
  </ParamField>
</ParamField>

## Methods

### mount

Mounts the QueryClient and subscribes to focus and online managers.

```typescript
queryClient.mount(): void
```

Internal mount count is tracked - subscriptions are only created on the first mount.

### unmount

Unmounts the QueryClient and unsubscribes from focus and online managers.

```typescript
queryClient.unmount(): void
```

Subscriptions are only removed when the mount count reaches zero.

### getQueryData

Imperative (non-reactive) way to retrieve data for a query key.

```typescript
queryClient.getQueryData<TQueryFnData, TTaggedQueryKey>(queryKey: TTaggedQueryKey): TInferredQueryFnData | undefined
```

<ParamField path="queryKey" type="QueryKey" required>
  The query key to get data for
</ParamField>

**Returns:** The cached data for the query, or `undefined` if not found.

**Example:**
```typescript
const data = queryClient.getQueryData(['todos'])
```

<Warning>
  Do not use this function inside a component, because it won't receive updates. Use `useQuery` to create a `QueryObserver` that subscribes to changes.
</Warning>

### setQueryData

Sets or updates data for a query key.

```typescript
queryClient.setQueryData<TQueryFnData, TTaggedQueryKey>(
  queryKey: TTaggedQueryKey,
  updater: Updater<TInferredQueryFnData | undefined, TInferredQueryFnData | undefined>,
  options?: SetDataOptions
): TInferredQueryFnData | undefined
```

<ParamField path="queryKey" type="QueryKey" required>
  The query key to set data for
</ParamField>

<ParamField path="updater" type="Updater" required>
  A value or function that receives the old data and returns the new data
</ParamField>

<ParamField path="options" type="SetDataOptions" optional>
  <ParamField path="options.updatedAt" type="number" optional>
    Custom timestamp for when the data was updated
  </ParamField>
</ParamField>

**Example:**
```typescript
queryClient.setQueryData(['todos'], (old) => [...old, newTodo])
```

### getQueriesData

Gets data for multiple queries matching the filters.

```typescript
queryClient.getQueriesData<TQueryFnData>(filters: QueryFilters): Array<[QueryKey, TQueryFnData | undefined]>
```

<ParamField path="filters" type="QueryFilters" required>
  Filters to match queries
</ParamField>

**Returns:** Array of tuples containing query keys and their data.

### setQueriesData

```typescript
queryClient.setQueriesData<TQueryFnData>(
  filters: QueryFilters,
```

### getQueryState

Gets the full state object for a query.

```typescript
queryClient.getQueryState<TQueryFnData, TError>(queryKey: QueryKey): QueryState<TQueryFnData, TError> | undefined
```

**Returns:** The query state including data, error, status, and metadata.

### fetchQuery

Fetches a query and returns a promise that resolves with the data.

```typescript
queryClient.fetchQuery<TQueryFnData, TError, TData, TQueryKey>(
  options: FetchQueryOptions<TQueryFnData, TError, TData, TQueryKey>
): Promise<TData>
```

<ParamField path="options" type="FetchQueryOptions" required>
  <ParamField path="options.queryKey" type="QueryKey" required>
    Unique key for the query
  </ParamField>
  
  <ParamField path="options.queryFn" type="QueryFunction" optional>
    The function that the query will use to request data
  </ParamField>
  
  <ParamField path="options.staleTime" type="number | 'static'" optional>
    Time in milliseconds after data is considered stale
  </ParamField>
</ParamField>

**Example:**
```typescript
const data = await queryClient.fetchQuery({
  queryKey: ['todos'],
  queryFn: fetchTodos,
})
```

### prefetchQuery

Prefetches a query and caches the result. Does not return the data.

```typescript
queryClient.prefetchQuery<TQueryFnData, TError, TData, TQueryKey>(
  options: FetchQueryOptions<TQueryFnData, TError, TData, TQueryKey>
): Promise<void>
```

**Example:**
```typescript
await queryClient.prefetchQuery({
  queryKey: ['todos'],
  queryFn: fetchTodos,
})
```

### ensureQueryData

Ensures that data for a query is available. If cached data exists, it's returned. Otherwise, the query is fetched.

```typescript
queryClient.ensureQueryData<TQueryFnData, TError, TData, TQueryKey>(
  options: EnsureQueryDataOptions<TQueryFnData, TError, TData, TQueryKey>
): Promise<TData>
```

<ParamField path="options.revalidateIfStale" type="boolean" optional>
  If true, stale data will trigger a background refetch
</ParamField>

```typescript
queryClient.fetchInfiniteQuery<TQueryFnData, TError, TData, TQueryKey, TPageParam>(

): Promise<InfiniteData<TData, TPageParam>>
```

<ParamField path="options" type="FetchInfiniteQueryOptions" required>
  <ParamField path="options.initialPageParam" type="TPageParam" required>

  </ParamField>
  
  <ParamField path="options.getNextPageParam" type="GetNextPageParamFunction" optional>
    Function to get the next page param

  <ParamField path="options.getPreviousPageParam" type="GetPreviousPageParamFunction" optional>
    Function to get the previous page param
  </ParamField>
  
  <ParamField path="options.pages" type="number" optional>
    Number of pages to fetch
  </ParamField>
</ParamField>

### invalidateQueries

Invalidates queries matching the filters, marking them as stale and potentially refetching them.

```typescript
queryClient.invalidateQueries<TTaggedQueryKey>(
  filters?: InvalidateQueryFilters<TTaggedQueryKey>,
  options?: InvalidateOptions
): Promise<void>
```

<ParamField path="filters" type="InvalidateQueryFilters" optional>
  <ParamField path="filters.queryKey" type="QueryKey" optional>
    Query key to filter by
  </ParamField>
  
  <ParamField path="filters.exact" type="boolean" optional>
    If true, only queries with the exact key will be matched
  </ParamField>
  
  <ParamField path="filters.refetchType" type="'active' | 'inactive' | 'all' | 'none'" optional>
    Which queries to refetch. Defaults to 'active'
  </ParamField>
</ParamField>

**Example:**
```typescript
await queryClient.invalidateQueries({ queryKey: ['todos'] })
```

### refetchQueries

Refetches queries matching the filters.

```typescript
queryClient.refetchQueries<TTaggedQueryKey>(
  filters?: RefetchQueryFilters<TTaggedQueryKey>,
  options?: RefetchOptions
): Promise<void>
```

<ParamField path="options" type="RefetchOptions" optional>
  <ParamField path="options.cancelRefetch" type="boolean" optional>
    If true, cancels any running queries before refetching. Defaults to true
  </ParamField>
  
  <ParamField path="options.throwOnError" type="boolean" optional>
    If true, throws errors instead of catching them
  </ParamField>
</ParamField>

### cancelQueries

Cancels ongoing queries matching the filters.

```typescript
queryClient.cancelQueries<TTaggedQueryKey>(
  filters?: QueryFilters<TTaggedQueryKey>,
  cancelOptions?: CancelOptions
): Promise<void>
```

<ParamField path="cancelOptions" type="CancelOptions" optional>
  <ParamField path="cancelOptions.revert" type="boolean" optional>
    If true, reverts to the previous data. Defaults to true
  </ParamField>
  
  <ParamField path="cancelOptions.silent" type="boolean" optional>
    If true, doesn't notify observers
  </ParamField>
</ParamField>

### resetQueries

Resets queries to their initial state and optionally refetches them.

```typescript
queryClient.resetQueries<TTaggedQueryKey>(
  filters?: QueryFilters<TTaggedQueryKey>,
  options?: ResetOptions
): Promise<void>
```

### removeQueries

Removes queries from the cache.

```typescript
queryClient.removeQueries<TTaggedQueryKey>(filters?: QueryFilters<TTaggedQueryKey>): void
```
